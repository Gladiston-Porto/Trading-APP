                    ⚠️ TROUBLESHOOTING RÁPIDO
═══════════════════════════════════════════════════════════════════════════════

"Docker command not found"
→ $ which docker
→ Se não achar, instalar Docker: https://docker.com/products/docker-desktop

"Image not found"
→ Script vai avisar se imagem não existe
→ Verificar se foi buildado: docker images | grep trading-app

"Docker login failed"
→ Verificar username/password do Docker Hub
→ Testar: docker login -u seu_username
→ Gerar token pessoal em: hub.docker.com → Account Settings

"Permission denied"
→ Dar permissão: chmod +x docker-hub-push.sh
